How to Start Claude Code
1

Open Finder

Click the Finder icon in your Dock (the smiling face icon at the bottom of your screen).

2

Navigate to your folder

In the left sidebar, click Documents. Find the folder called claude code test

3

Open Terminal here

Right-click (or Control-click) the folder. In the menu that appears, select the second to last option: "New Terminal at Folder"

4

Start Claude Code

A Terminal window opens. Type claude and press Enter. You are now inside Claude Code.

5

Run your first command

Type /briefing to see your daily intelligence. Or type /command-center to see everything available.

Your 5 Daily Commands
📊
/briefing
Your morning intelligence. Revenue, calendar, emails, leads, and flags. Start every day here.
🧠
/board-meet [question]
Ask your advisory board anything. Pricing, strategy, growth, hard decisions.
Example: /board-meet Should I raise my membership to $297?
✍️
/content-engine [topic]
Full content suite from one topic. Blog, social posts, video script, newsletter, visuals. Deep research first.
Example: /content-engine overcoming fear of failure using neuroscience
🔊
/voice [paste text]
Rewrites any text in your brand voice. Paste an email, caption, or paragraph.
🚀
/launch [product]
Complete launch system. ICA, offer, funnel, emails, content, ads, visuals. 10 builds, step by step.
Example: /launch NeuroGym Mental Fitness Membership
Quick Tips
↩️
Resume a previous chat: Type claude --resume in Terminal to pick up where you left off.
✨
Start fresh: Type claude in Terminal for a new session.
💬
Just ask in plain English. You do not need special syntax. Describe what you want and Claude figures out the right command.
💾
Everything saves automatically. Your business profile, voice, learnings, and board all persist between sessions.
📋
See all commands: Type /command-center inside Claude Code for the full list.
If Something Goes Wrong
"command not found: claude"
Close the Terminal window. Open a new one from your folder (Step 1-3 above). Try again.
Claude is stuck or not responding
Press Ctrl+C (hold Control and press C). Then type claude to start a new session.
Accidentally closed Terminal
No problem. Follow Steps 1-4 above. Type claude --resume to continue where you left off.
Need help with anything else
WhatsApp Muhammad. Or schedule a follow-up session.
